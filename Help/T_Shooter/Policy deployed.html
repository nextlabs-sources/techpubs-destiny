Is this policy actually deployed?

If the policy is not being triggered, it may be because it is not deployed at all on that enforcer. To confirm that the policy is deployed:

  • For policies on either file server or desktop enforcers, you can open Policy Author, select the policy, then click the Deployment Status command from the Actions menu. This window has two tabs, listing all File Server Enforcer and Desktop Enforcer hosts. On the appropriate tab, select the enforcer you are troubleshooting, and all policies and components currently deployed on it will display in the Contents list.
