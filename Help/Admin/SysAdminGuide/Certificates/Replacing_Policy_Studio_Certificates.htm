Replacing Policy Studio Certificates

One additional configuration step must be performed on all devices where Policy Studio is installed.

  1. Overwrite the existing Policy Studio keystore, policyAuthor-key­store.jks at:  <installdir>Policy Studio\plugins\com.next­labs.policystudio.dependencies\client-pf.jar\security\policyauthor-keystore.jks.(You must open the client-pf.jar file using 7-zip and overwriting the original policyAu­thor-keystore.jks file in the security folder.)

  2. Restart Policy Studio.
