Overview of Procedures

The general procedure for replacing certificates include the following steps:

  1. Generate a new keystore.

  2. Generate a new certificate from the keystore and have it signed by an external authority.

  3. Import the certificates into the pertinent truststore files.

  4. Overwrite old truststore and keystore files in all relevant locations.

For example, the truststore for the Control Center Web Services (referred to as DCC) must include certificates for all the components the Control Center com­municates with: the Policy Controller, Enrollment Manager, Policy Studio, and Policy Server tools.

Command Syntax Conventions

All procedures in this chapter use Java Keytool. In all commands presented in this chapter, values that are user-defined are presented in angle brackets (<>). Values that are not in angle brackets are not user-defined and must be entered exactly.
