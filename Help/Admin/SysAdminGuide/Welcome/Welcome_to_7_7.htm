What’s New in this Release

This release provides the following new features:

·        Access control on monitors and alerts, which is determined by the Reporter Administration and Reporter Analysis applications in the role definitions.

Users assigned a role that includes the Reporter Administration applica­tion can grant read-only access to the monitors they create to other users.

Users assigned a role that includes the Reporter Analysis application can view the list and definitions of monitors to which they are granted access. These users can also view alerts that are generated by the monitors to which they have been granted access.

·        Support for locale-specific dates when configuring Control Center options that take a date value.
