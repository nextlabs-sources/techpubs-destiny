Welcome to the product documentation for Control Center Reporter. This user guide provides information on using Reporter, the web application you use to generate reports on how information is being used in your environment, and on the effectiveness of your risk management policies. ·        Introducing Reporter

·        Using the Dashboard

·        Working with Reports

·        Working with Monitors and Alerts

What’s New in this Release

This release provides the following new features:

·        Two types of Reporter users. Users granted Reporter administrative privi­leges have access to all Reporter features and functionality. Users granted Reporter analysis privileges can only run and view reports that administra­tors share with them. Previously, Reporter users either had complete access to all functionality, or no access to Reporter.

·        Capability for administrators to make a report publicly available to all Reporter users, or to select users and groups. Previously, users could des­ignate reports they created as either private or publicly available to all users.
