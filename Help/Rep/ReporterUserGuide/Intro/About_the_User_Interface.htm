About the User Interface

Reporter is organized into three tabs: Dashboard, Reports, and Monitoring. You access each by clicking the associated links. By default, the Dashboard tab dis­plays when Reporter opens. Only users granted administrator permissions, or access to monitors, can access the Monitoring tab. The figure below shows a par­tial view of a sample dashboard.

Note: If your system has any optional Application modules installed, such as SharePoint Governance or Endpoint Data Protection, your Reporter might display additional tabs to represent the reporting features of those applications. The content and functions of these additional tabs are described in the respective Application user’s guides.

 

Dashboard tab (partial view)

Dashboard Tab

The Dashboard tab displays charts and tables representing a set of predefined queries that are updated automatically with fresh data from the Activity Jour­nal. For more information about using the Dashboard, see Using the Dashboard.

Reports Tab

The Reports tab is where you define and run reports, save them for future reuse and, optionally, share them with other users. For more information about defin­ing reports, see Working with Reports.

Monitoring Tab

The Monitoring tab is where you define monitors to receive notifications (alerts) about specific policy enforcements, and view alerts triggered by the monitors. Only users granted administrator permissions, or access to monitors, can access this tab. For more information about creating monitors, see Working with Monitors and Alerts.

Banner

The banner at the top of the web page displays version information and links.

Release Version

The release version of the Reporter web application appears under the logo in the upper right corner. The number in parentheses is the build number for that release.

Note that this build number reflects not only Reporter per se, but also the ver­sion of the Control Center itself. You may be asked to provide this information if you contact NextLabs technical support.

Logout

The Logout link logs the current user out of Reporter. You can also log out by closing your web browser, or the active tab in your web browser.

Change Password

The Change Password link allows you to change your login password. Changes made here are saved in Administrator, as a property of the user as defined there.

Help

The Help link displays an HTML version of this manual in your default web browser.
