Opening Reporter

You access Reporter using a web browser.

Note: If you are using Firefox, you cannot access Reporter unless you enable TLS 1.0 (under Tools, Options, Advanced).

To open Reporter, perform the following procedure:

  1. In your web browser, enter the URL where Reporter is installed in your network. The URL is in the following form:

    2. https://<hostname>/reporter

    If you do not know the URL, ask your system administrator.

  2. In the Login screen, shown in the following figure, type a valid user name and password. If you have not been assigned a user name, see your system administrator.

    3. You can also use the built-in login name Administrator, along with the password that was set for the Administrator account during Control Center installation.

     

    Reporter Login

  3. Click Login. The Reporter’s main window appears, as shown in Figure 1-2.
