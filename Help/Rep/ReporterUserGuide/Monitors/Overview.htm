Overview

In Reporter, you use the Monitoring tab to create policy monitors and to view alerts triggered by those monitors.When you click the Monitoring tab, the Alerts Overview page appears. It displays a list of alerts, if any, in the order they were raised, with the latest on top. The figure below shows an example of this page with no alerts displayed. This is the default view before any monitors are defined or before any alerts are triggered.

If there are alerts, you can take the following actions:

·        Filter alerts by specified criteria, including time period, monitor, and tag name and value

·        View alerts aggregated by tag, monitor, or time, and presented in chart format

·        Dismiss any alert from the list

·        Delete any alert from the list

 

Alerts Overview page

As the figure above shows, two subtabs appear below Monitoring: Alerts and Monitors. Click Monitors to view the list of defined monitors, if any, and to cre­ate a new monitor. The following figure shows an example of the Monitors page with no monitors displayed. This is the default view before any monitors are defined.

 

Monitors page

You can perform the following tasks in the Monitors tab:

·        Create monitors

·        Edit monitors

·        Deactivate and activate monitors

·        Delete monitors
