Group By User Reports

When we group our data by user and re-run the report, we see the following graph. As we have noted, the four policies were each triggered by a different user, and so the graph shows four bars—each representing one user. Each is labeled with a user name. In this example the bars are the same height, since each of the four users triggered a policy one time.

 

Group By User Report

All things being equal, in a live environment there will generally be little differ­ence among users’ rates of triggering policies. In cases where a big difference does arise—when one user shows a significantly disproportionate rate of trigger­ing policies (especially Deny policies)—then the situation will likely merit closer investigation.
